# PDF Export

## Local
```bash
pdflatex resume.tex
```

If references are used:
```bash
pdflatex resume.tex && pdflatex resume.tex
```

## Overleaf
1. Upload `resume.tex` and supporting files.
2. Select pdfLaTeX compiler.
3. Build and download PDF.

## Notes
- Keep to one page.
- Verify ATS readability by selecting/copying text from exported PDF.
